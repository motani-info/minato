# 実装計画: UIデザイン洗練化

## 概要

「みなトレ」アプリのビジュアルデザインを段階的に洗練する。Tailwind CSS 4 のデザイントークン定義から始め、新規コンポーネント作成、既存コンポーネントのリファクタリング、App.tsx の全面改修の順に進める。各ステップはビルド可能な状態を維持しながらインクリメンタルに実装する。

## 備考

- 各タスクは前のタスクの成果物に依存するため、順番通りに実装すること
- Framer Motion は既に導入済み（framer-motion@12.38.0）のため追加インストール不要
- 外部ライブラリの追加は行わず、CSS + Framer Motion で全エフェクトを実装する
- index.css のカスタムCSS削除は段階的に行い、各ステップでビルドが通ることを確認する
